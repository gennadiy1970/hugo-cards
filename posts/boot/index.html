Скорость загрузки страницы - важнейший показатель работы сайта.

Google опубликовал исследование из которого следует, что пользователь переключается на другой сайт, если не дождался появления контента в течение 2,5 секунд.

По этой причине Google ранжирует сайты при выводе поискового запроса в том числе и по скорости загрузки страницы. Все это требует от FrontEnd-а постоянного внимания к вопрсу скорости загрузки.

Два важных параметра ограничивающие скорость загрузки страницы

  1. Максимальное количество параллельных загрузок браузера = 17 соединений.
  2. Максимальное количество параллельных загрузок браузера с одного сервера зависит от браузера, но в среднем 6-8 соединений.

Проверить для текщей версии браузера можно по ссылке Network Tests For The Browser

Проверить скорость загрузки сайта можно в Inspector браузера, вкладка Network или hot-key Ctrl + Shift + E для FireFox.
