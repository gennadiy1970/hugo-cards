Работа с директориями

October 27, 2018

Keys
Behavior
mkdir app
создаст директорию app
mkdir app{1..50}
создаст 50 директорий с именами от app1 до app50
mkdir -p src/{css,img,js}
создаст директорию src с вложенными директориями css, img, js
mkdir -p year{2015..2020}/month{1..12}
создаст 6 директорий и в каждой по 12 директорий (year2015/month1)
mkdir class-{A..D}
создаст 4 директории class-A, class-B, class-C, class-D
mkdir -p `date '+%Y-%m-%d'`/{jpg,raw}
создаст директорию вида 2018-10-15 с вложенными директориями jpg, raw
pwd
выведет строку с текущей директорией
cd src
перейдет в директорию src или сообщит об отсутствии такой директории
cd ../
перейдет в родительскую директорию (на 1 уровень выше)
cd ../../
перейдет в родительскую директорию (на 2 уровеня выше)

Скопруйте этот текст

alias setdir='touch index.html | mkdir ./{css,js,img,assets} && touch ./{js/app.js,css/style.css}'

Вставьте в файл C:\Program Files\Git\etc\profile.d\aliases.sh Перегрузите gitbash и теперь команда setdit будет создавать директории и файлы в директории
