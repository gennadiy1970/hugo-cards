Работа с файлами

October 9, 2018

Keys
Behavior
touch index.html
создаст пустой файл index.html
echo > index.html
создаст пустой файл index.html
echo 'Hello World' > index.html
создаст файл index.html c текстом 'Hello World'
cat index.html
покажет в терминале текст из файла index.html
echo 'new line text' >> index.html
добавит текст 'new line text' в файл index.html с новой строки
diff index.html home.html
выведет в терминал текст, описывающий как изменить первый файл для тождества со вторым файлом в виде кода (2,4c2,4) и текста.
